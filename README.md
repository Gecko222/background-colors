## Requirements

Node.js, npm

## Installation

```
> yarn install
```

## Build application for production

```
> yarn run build
```

## Development

### Webpack dev server

```
> yarn start
```

### Tests

```
> yarn test
```

### Eslint

```
> yarn lint
```
